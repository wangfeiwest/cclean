CClean增强版 - 杀毒软件白名单设置说明
================================================

如果您的杀毒软件误报本程序为病毒，请按以下步骤操作：

1. Windows Defender 白名单设置：
   - 打开 Windows 安全中心
   - 选择"病毒和威胁防护"
   - 点击"管理设置"（病毒和威胁防护设置下）
   - 向下滚动到"排除项"，点击"添加或删除排除项"
   - 点击"添加排除项" → "文件"
   - 选择 CClean增强版_v3.0.exe 文件

2. 其他杀毒软件：
   - 360安全卫士：安全防护中心 → 信任与阻止 → 添加信任文件
   - 火绒安全：设置 → 信任区 → 添加文件到信任区
   - 卡巴斯基：设置 → 其他 → 威胁和排除 → 排除 → 添加

3. 为什么会被误报？
   - PyInstaller 打包的程序特征容易触发启发式检测
   - 程序需要管理员权限访问系统文件
   - 程序会批量删除文件，行为类似清理工具

4. 程序安全性保证：
   - 源代码开放，可自行检查
   - 只清理临时文件和缓存，不修改重要系统文件
   - 包含安全检查机制，避免误删重要文件

5. 获得数字签名（推荐）：
   - 购买代码签名证书为exe文件签名
   - 这样可以大大减少误报率

如有疑问，请检查源代码确认程序安全性。
